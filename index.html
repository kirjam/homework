HTML (от англ. HyperText Markup Language - "язык гипертекстовой разметки") - стандартизированный язык разметки документов во вселенной Всемирной паутине.
Большинство веб-страниц содержат описание разметки на языке HTML (или XHTML). Язык HTML интерпретируется браузерами; полученный в результате интерпретации
форматированный текст отображается на экране монитора компьютера или мобильного устройства.

CSS (от англ. Cascading Style Sheets - каскадные таблицы стилей) - формальный язык описания внешнего вида докумта, написанного с использованием языка разметки.
